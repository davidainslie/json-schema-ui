Dashboard

 

Examples of defining data in JSON schema which is presented in web page(s).
Web forms can then be auto constructed from data definitions.
The presentation of data is handled by HTML templates with the help of Angularjs.
A JSON schema simply declares data along with its rules/validations and dependencies allowing for complex presentations. The examples, and indeed this application, works across the main browsers Chrome, Safari, Firefox and IE.
The application also works across devices - currently tested on iPad and iPhone.
